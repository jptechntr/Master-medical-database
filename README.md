# Master Medical Database

## Description
A comprehensive database management system designed for managing medical records and information, enabling efficient data handling for healthcare providers.

## Features
- User-friendly interface for easy navigation
- Secure login and user authentication
- CRUD operations for managing patient records
- Search functionality for quick access to medical information
- Reporting tools for analyzing patient data

## Project Contents
- `medicines_master_dataset.csv` - Master dataset of medicines
- `index.html` - Interactive web interface for viewing medicines

## How to Access
Your medical database is now live on GitHub Pages at:
**https://jptechntr.github.io/Master-medical-database/**

## Installation
1. Clone the repository: `git clone https://github.com/jptechntr/Master-medical-database.git`
2. Navigate to the project directory: `cd Master-medical-database`
3. Open `index.html` in your browser
